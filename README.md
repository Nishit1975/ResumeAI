# ResumeAI – Smart Resume Builder

A modern resume builder built with HTML, CSS and JavaScript.

## Features

- Live Resume Preview
- 3 Resume Templates
- Dark / Light Mode
- Auto Save using LocalStorage
- PDF Export
- Responsive Design
- Profile Photo Support (Template 3)

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6+)
- LocalStorage API
- Browser Print API
